Let's start!

What is Xorg?

Xorg is a app for opening /dev/random and /dev/random is writing screen bytes in /dev/fb0, like is a support for opening DE (Desktop Enviroment)/WM (Window Manager), GUI if you don't understand.

What is Wayland?

This too like Xorg but it's small other..

PLEASE NOT RUN WAYLAND ON NVIDIA (or you videocard is will burn) or with small chance your videocard is not will burn

Wayland DE's or WM's: Hyprland (my best wm on wayland), GNOME (it's a desktop enviroment), Sway (analog i3 but on wayland), Xfce (Wayland) (it's xfce but on wayland), Plasma (Wayland) (it's plasma but on wayland too)

How to install Xorg?

Okay install Xorg is easy :))), enter in tty:

sudo pacman -S xorg-server xorg-xinit

Or if you don't have sudo, login in root:

su

And enter command but without sudo

What is SDDM?

SDDM (Simple Desktop Display Manager) - It's a very good display manager, display manager is a app for select DE's/WM's and select user and enter password and login

How to install SDDM?

Enter in your tty:

pacman -S sddm

And after install enter this:

systemctl enable --now sddm

And done! You have installed SDDM!

What is sudo?

sudo (Root Do/Su Do) - Is a root do but without login in root, it's good app!

How to install sudo?

Enter in your tty:

pacman -S sudo

After install you need open /etc/sudoers

And in end file found this line where:

# root ALL=(ALL:ALL) ALL

# %wheel ALL=(ALL:ALL) ALL

# %sudo ALL=(ALL:ALL) ALL

# ALL ALL=(ALL:ALL) ALL

Delete all '#' from lines

And you have working and installed sudo!

What is XFCE?

XFCE - It's a easy and simple desktop enviroment for newbie's of Linux

How to install XFCE?

Enter in tty this command:

sudo pacman -S xfce4

And you have installed XFCE!

How to create user?

Create user easy!

Enter this in your tty:

useradd -m -G users,wheel,audio,video,tty name_for_new_user

And you create a new user!

TTY Browser?..

Yes! TTY Browser is exist!

Browsers in TTY: Lynx (old text browser and good), Links (good too but without styles), Elinks (links but with styles but small strangest..)

How to install TTY Browsers?

Lynx:

sudo pacman -S lynx

Links:

sudo pacman -S links

Elinks:

sudo pacman -S elinks

And you have installed TTY Browser!

How to install Wayland?

Install wayland easy!

Enter in terminal/console/tty this:

sudo pacman -S wayland xwayland

Wayland - main app

Xwayland - for running Xorg apps on Wayland
